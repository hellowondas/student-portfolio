## 📋 Overview

The goal was to design a fully responisve, accessible and pixel-perfect multi-page website that serves as a personal academic portfolio and student management platform using HTML, CSS, and JavaScript.


| Desktop | Tablet | Mobile |
| ------- | ------ | ------ |

## 🚀 Features

- Fully responsive web page that displays on different devices and screen sizes.
- Clean, semantic, accessible webpage and user-friendly
- Standard mobile first design practice
- Optimized local images for fast loading
- No frameworks

## 🛠️ Built With

- HTML5
- CSS3 (no frameworks)

## 📦 Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. All assets are local; no build step is required.
